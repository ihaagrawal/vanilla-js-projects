# 🌍 Timezone Viewer

A simple web application built using **HTML, CSS, and JavaScript** that allows users to check the **current time of any city**.

---

## 🚀 Features

* Input any city name
* Displays current time for the entered city
* Clean and minimal UI
* Beginner-friendly project
* Easy to extend with APIs

---

## 📂 Project Structure

```
Timezone-Viewer/
│── index.html
│── style.css
│── script.js
```

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling and layout
* **JavaScript (Vanilla JS)** – Logic and DOM manipulation

---

## 📖 How It Works

1. User enters a city name
2. Clicks on **"Current Time"** button
3. JavaScript fetches or calculates the current time
4. The result is displayed inside the content section

> ⚠️ Note: To get accurate city-based time, you typically need:
>
> * `Intl.DateTimeFormat` (basic approach), or
> * A timezone API (recommended for real-world apps)

---

## ▶️ How to Run

1. Download or clone the project
2. Open `index.html` in your browser
3. Enter a city name and click **Current Time**

---

## 💡 Example

```
Input:
City: London

Output:
Current Time: 10:30 AM
```
