# 🚀 Vanilla JavaScript Projects

A collection of beginner-to-intermediate **web projects built using HTML, CSS, and Vanilla JavaScript**.
This repository showcases core JavaScript concepts, DOM manipulation, keyboard events, and interactive UI development.

---

## 📌 Projects Included

### 🧮 BMI Calculator

* Calculates Body Mass Index using height and weight
* Displays BMI value and health category

---

### 🎨 Color Changer

* Changes background color dynamically
* Demonstrates DOM manipulation and event handling

---

### 🌈 Unlimited Colors

* Continuously changes background color every second
* Start/Stop functionality using `setInterval` and `clearInterval`
* Prevents multiple intervals (important JS concept)

---

### ⌨️ Keyboard Key Checker

* Detects single and multiple key presses
* Displays key combinations (e.g., `Ctrl + Shift + A`)
* Shows ASCII values for character keys
* Stores and displays the **last key combo** even after release
* Demonstrates advanced event handling (`keydown`, `keyup`)

---

### 🌍 Timezone Viewer

* Displays current time based on user input
* Introduces working with time and APIs

---

### 🎯 Guess The Number

* Interactive number guessing game
* Includes input validation, game logic, and state management

---
