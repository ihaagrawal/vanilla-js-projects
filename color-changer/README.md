# 🎨 Color Changer

A simple and interactive **Color Changer Web App** built using **HTML, CSS, and JavaScript**.
It allows users to change the background color of the page by clicking on color buttons.

---

## 🚀 Features

* Instant background color change
* Multiple predefined color options
* Clean and minimal UI
* Easy to extend with more colors
* Beginner-friendly JavaScript project

---

## 📂 Project Structure

```
Color-Changer/
│── index.html
│── style.css
│── index.js
```

---

## 🛠️ Technologies Used

* **HTML5** – Page structure
* **CSS3** – Styling and layout
* **JavaScript (Vanilla JS)** – DOM manipulation and interactivity

---

## 📖 How It Works

1. The page displays multiple color buttons
2. Each button has a color name as its text
3. When a button is clicked:

   * The `handleClick()` function is triggered
   * It reads the button's text (color name)
   * Sets the background color of the page using:

```
document.getElementById("body").style.backgroundColor = color;
```

4. The background updates instantly

---

## ▶️ How to Run

1. Download or clone the project
2. Open `index.html` in your browser
3. Click on any color box to change the background

---

## 💡 Example

```
Click: "red"

Result:
Background color changes to red
```
